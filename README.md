# myGoodlife Club — Loyalty CRM

Internal Loyalty CRM platform for the myGoodlife Club programme at Goodlife Pharmacy.

## Overview

Built to replace Excel-based workflows with a structured, data-driven system for the marketing team. Supports member segmentation, campaign building, and CSV export for outbound campaigns.

## Tech Stack

- **Frontend**: Next.js + Tailwind CSS
- **Database & Auth**: Supabase (PostgreSQL)
- **Hosting**: Vercel

## Development

```bash
npm install
npm run dev
```
